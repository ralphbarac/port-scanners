# Port Scanners in various languages

Running the applications

## Python Application

```
python python_app.py
```

When prompted, enter a host (ie. www.theathletic.com) and a port range (ie. 50-80).

## Elixir Application

```
elixir elixir_app.exs
```

When prompted, enter a host (ie. www.theathletic.com) and a port range (ie. 50-80).
